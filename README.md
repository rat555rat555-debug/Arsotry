# AR Poster Web Ready

เว็บนี้คือเว็บ HTML สำหรับสแกนภาพโปสเตอร์แล้วแสดงวิดีโอทับบนภาพ

## ไฟล์สำคัญ

- `index.html` = หน้าเว็บหลัก
- `style.css` = หน้าตาเว็บ
- `script.js` = ระบบเปิดกล้อง / ตรวจจับภาพ / เล่นวิดีโอ
- `targets/poster.mind` = ไฟล์ภาพเป้าหมายที่ต้องสร้างจากโปสเตอร์จริง
- `videos/sample.mp4` = วิดีโอที่จะแสดงเมื่อสแกนเจอภาพ

## วิธีเปลี่ยนเป็นของตัวเอง

1. เตรียมภาพโปสเตอร์ เช่น `poster.jpg`
2. เอาภาพไปแปลงเป็นไฟล์ `.mind`
   - ใช้เครื่องมือของ MindAR Image Compiler
   - ผลลัพธ์ต้องชื่อ `poster.mind`
3. เอาไฟล์ `poster.mind` ไปวางในโฟลเดอร์ `targets`
4. เอาวิดีโอของตัวเองไปวางในโฟลเดอร์ `videos`
5. ตั้งชื่อวิดีโอเป็น `sample.mp4`
6. เปิดเว็บผ่าน `https://` หรือ `localhost`

## ข้อสำคัญ

- เปิดไฟล์แบบกด `index.html` ตรง ๆ อาจใช้กล้องไม่ได้
- ต้องเปิดผ่านเซิร์ฟเวอร์ เช่น GitHub Pages, Netlify, Vercel หรือ localhost
- ภาพเป้าหมายควรมีรายละเอียดเยอะ ไม่ควรเป็นภาพโล่ง ๆ สีเรียบ ๆ
- วิดีโอแนะนำเป็น `.mp4`
- มือถือ iPhone/Android ต้องกดปุ่มเริ่มก่อนเพื่อให้วิดีโอเล่นได้

## ทดสอบบนคอมแบบง่าย

ถ้ามี Python ให้เปิดโฟลเดอร์นี้แล้วรัน:

```bash
python -m http.server 8000
```

จากนั้นเปิด:

```text
http://localhost:8000
```

## โครงสร้างระบบจริง

Creator:
- อัปโหลดภาพโปสเตอร์
- อัปโหลดวิดีโอ
- แปลงภาพเป็น `.mind`
- บันทึกโปรเจกต์

Viewer:
- เปิดลิงก์
- อนุญาตกล้อง
- สแกนภาพ
- แสดงวิดีโอทับบนโปสเตอร์
